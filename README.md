# 🚌 オレンジゆずるバス・ナイトツアー(赤ルート) Ver 5.0 (MapLibre GL版)

## 🗺️ Ver 5.0 の大改修: ベクター地図への移行
地図エンジンを Leaflet から **MapLibre GL + ベクタータイル(OpenFreeMap・無料/キー不要)** に移行しました。
- **文字情報なしの自作ナイトスタイル**: 地名・道路名等のラベルを一切描画せず、道路(3階級の光の帯)・建物・水面・緑地だけのミニマル構成です
- **建物の3D押し出し**: OSMの建物形状を高さ情報で立体化。高い建物ほど淡く光る配色です
- **斜めカメラ**: `pitch: 55` の見下ろしアングルで疑似3D表現(0〜60で調整可)
- **中間ズーム**: `fixedZoom: 18.5` のような小数指定が可能になりました
- **滑らか追従**: カメラはバスに直接固定せず、イージング補間で「ふわっと遅れて」追従します。`cameraEase`(既定2.8)を小さくするとより余韻のある動き、大きくするとキビキビした動きになります

## 🌙 夜の絵本テーマについて
青い環境光・流れる霧・画面ふちのぼかし(被写界深度風)・ルート沿いの暖色の街路灯・光のにじむルートラインを重ねて、絵本のようなやさしい夜景を表現しています。バスはぬいぐるみ風のデフォルメSVGで、ぽよぽよと弾みながら走ります。
- `nightTheme: false` にすると従来の昼マップに戻せます
- `dreamyDof: false` でふちのぼかしを無効化できます(動作の重い端末向け)
- `lampIntervalM` で街路灯の間隔を調整できます
- 運行開始時の音声は画面下部に字幕(`voiceText` の文章)として一文字ずつ表示されます

xlsxリストからルート・イベントを読み込み、OpenStreetMap上でバスが1周30分で周回するアプリです。スマートフォン画面を基準にUIを構成しています。

## 📂 ファイル構成
| ファイル | 役割 |
|---|---|
| `route-data.js` | ルート・イベントの全データ(xlsxから自動生成) |
| `tools/generate_route_data.py` | **xlsx → route-data.js 変換スクリプト** |
| `app.js` / `index.html` / `style.css` | アプリ本体(編集不要) |
| `bus.png` | (旧)バス画像。現在はSVG製ぬいぐるみバスを使用しており未参照です |
| `cafe.png` | イベント目的地のアイコン(xlsxのicon列で指定・幅は `eventIconWidth` で調整) |
| `voice.wav` | 運行開始時に再生される音声(こんにちは 0.0.2) |
| `bgm.mp3` | 任意のループBGM。未設置でも通知は表示されません(WAVを使う場合は `bgmFile` を変更) |

## 🔄 リスト(xlsx)を更新したら
```
python3 tools/generate_route_data.py 新しいリスト.xlsx route-data.js
```
これだけで反映されます。カテゴリの扱いは以下の通りです。
- `busstop` … バス停。専用アイコンを表示し、5秒間停車します
- `corner` … 経路の角。地図には表示されず、ルートの形状にのみ使われます
- `event` … イベントの発動地点。**地図には表示されません**。バスが半径30m(`triggerRadius`)に入ると発動します
- `rout` 列 … 発動時に点線で順に結ばれる経路
- `event point` + `action: luminescence` … 点線の終点。緑の輝く輪が波紋状に広がり、`icon` 列の画像が表示されます

## 🎛️ UIのご案内
- **▶ 運行スタート** … 運行開始と同時に音声(voice.wav)が流れ、BGM(bgm.mp3)がループ再生されます。カメラはバスを中心に自動追従します
- **🎥 追従ボタン** … カメラ追従のオン・オフ。地図を手で動かすと自動的に解除されます
- **✉️ 投稿モード** … 地図をタップすると座標が「34.83364773416608, 135.4885148298627」形式(Googleマップと同形式)で表示されます。「コピー」またはメールフォームの起動による投稿が可能です。送信先アドレスは `route-data.js` の `postEmail` を変更してください
- **🗺️ 全体表示** … ルート全体が収まる縮尺に戻ります
- **倍速セレクタ** … 動作確認用です。本番では「等速(30分)」をご利用ください

## 🚀 起動方法
フォルダ内で `python3 -m http.server` などを実行し、`http://localhost:8000` を開いてください(file:// 直接ではスマホ・音声まわりが正しく動作しない場合があります)。
